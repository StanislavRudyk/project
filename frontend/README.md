# Frontend Architecture Documentation

Стек технологий:
- Сборщик: Vite 
- Фреймворк: React + TypeScript 
- Работа с API (данными): TanStack Query (React Query)
- Глобальный стейт: Zustand 
- Роутинг: React Router v6.
- Стилизация: Vanilla CSS (CSS Modules)

## Структура папок

```text
frontend/
├── public/                # Статические файлы (иконки, картинки), доступные напрямую по URL
└── src/
    ├── api/               # Глобальные настройки сети
    │   └── apiClient.ts   # Инстанс axios с базовым URL и перехватом токенов (Interceptors)
    │
    ├── assets/            # Картинки, шрифты и другие медиа-ресурсы для импорта в код
    │
    ├── components/        # Общие переиспользуемые UI-компоненты
    │   ├── ui/            # "Глупые" компоненты (Кнопки, Инпуты, Модалки)
    │   │   ├── Button/
    │   │   │   ├── Button.tsx
    │   │   │   └── Button.module.css
    │   │   └── Input/
    │
    ├── features/          # Изолированные фичи (бизнес-логика, разбитая по доменам)
    │   ├── books/         # Пример: все, что связано с книгами
    │   │   ├── api/       # Запросы к /api/books
    │   │   ├── components/# Компоненты, специфичные только для книг 
    │   │   └── store/     # Локальный стейт фичи 
    │   └── auth/          # Все, что связано с авторизацией 
    │
    ├── hooks/             # Кастомные глобальные React хуки 
    ├── layouts/           # Шаблоны страниц  для роутера
    │
    ├── pages/             # Компоненты-страницы, которые собирают фичи и компоненты вместе
    │   ├── Home/
    │   ├── Catalog/
    │   └── Profile/
    │
    ├── router/            # Настройки маршрутизаци
    │
    ├── store/             # Глобальный стейт приложения 
    │
    ├── styles/            # Глобальные стили
    │   ├── global.css     # Базовый сброс стилей и общие классы
    │   └── variables.css  # Design System: CSS-переменные (цвета, шрифты, отступы)
    │
    ├── utils/             # Вспомогательные функции
    │
    ├── App.tsx            # Корневой компонент 
    └── main.tsx           # Точка входа, подключение провайдеров 
```
